# GitHub Introduction

## GitHub Vocabulary
- git
- GitHub
- remote
- local
- branch
- clone
- repo (repository)
- GUI
- markdown
- diff (difference in the local and remote)

### GitHub Commands 
- git status 
- git add <filename>
- git commit -m "description of changes made"
- git push origin main

## Keyboard Shortcuts 
- command + s (safe)
- command + x (cut)
- command + v (paste)
- command + "spacebar" (opens spotlight search)
- command + r (refresh)
- command + c (copy)
- command + z (undo)
